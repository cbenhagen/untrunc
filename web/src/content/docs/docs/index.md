**rsv.repair** turns interrupted Sony `.rsv` recordings into a normal video file you can open anywhere.

If recording stops suddenly (dead battery, memory card removed, or a crash), the camera may save a `.rsv` file. Your footage is usually still inside; the file just doesn’t play in a normal app yet.

**Some Sony cameras** (e.g. VENICE, FS6, FS7, A7S Mark III) can **recover unfinished clips in playback mode** on the camera first. See [Check your camera for recovery options](/docs/what-is-rsv/#check-your-camera-for-recovery-options). If that doesn’t apply or doesn’t work, **rsv.repair** can rebuild a file on your computer: you add a **working clip from the same Sony camera**, then your `.rsv` file, choose where to save, and the tool writes a playable video. Nothing is uploaded.
